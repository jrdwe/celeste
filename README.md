## n-body physics simulation

A CUDA implementation of the all-pairs n-body simulation. An astrophysics simulation of orbiting bodies.

![Example Simulation](output/result.gif)

## Usage 

Command line usage: 
```
make celeste
./celeste <filename>
```

Generate video from ppm files via ffmpeg
```
ffmpeg -framerate 60 -i frames/frame%07d.ppm output/result.mp4
```

## Input data

Initial data generated by `Nemo - A Stellar Dynamics Toolbox` from https://teuben.github.io/nemo

```
mkking out=- nbody=15000 W0=7 | snapprint - x,y,z,vx,vy,vz,m > simulation.dat
```

All results generated in output/ were done on a nvidia turing gpu on an g4dn.xlarge ec2.
